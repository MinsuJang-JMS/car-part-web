## 1. 배경 및 목적

### 1-1. 프로젝트 개요
차량 부품을 판매하는 B2B/B2C 통합 웹사이트.
개인 고객과 사업자 고객을 구분하여 맞춤형 가격 정책을 적용하고,
차대번호(VIN) 기반 차량 조회로 호환 부품을 안내한다.
국산 및 수입차종 부품(도·소매) 정품·OEM·애프터마켓 전문.

### 1-2. 기술 스택 (완전 무료)
| 역할 | 기술 |
|------|------|
| 프론트엔드 | Next.js 16 (App Router) + TypeScript |
| 스타일링 | Tailwind CSS |
| 인증 | Firebase Auth (이메일) |
| 데이터베이스 | Firebase Firestore |
| 이미지 저장 | Firebase Storage |
| 배포 | Vercel (무료) |

### 1-3. 핵심 기능
| 기능 | 설명 |
|------|------|
| 사용자 유형 구분 | 개인 / 사업자 회원 분리 |
| 사업자 가입 검수 | 담당자가 사업자 여부 확인 후 승인 |
| 등급별 가격 표시 | 개인 / 사업자 A·B·C 등급별 가격 차등 |
| 카테고리 브라우징 | 브랜드 / 전체상품 / 부품 카테고리 탐색 |
| 홈 큐레이션 | 베스트 상품 / 신상품 섹션 노출 |
| 홈 비주얼 강화 | 수입/국산차 브랜드 쇼케이스(Firebase 동적), 풀스크린 히어로 |
| 브랜드 카테고리 분류 | 외제차 정품 / 국산차 정품 / OEM 3종으로 구분 관리 |
| 브랜드 이미지 관리 | 어드민에서 브랜드별 차량 사진 업로드/URL 등록 |
| 상품 관리 | 등록·활성화/숨기기·삭제 |
| 회사 소개 페이지 | /about — 소개글, 특징, 연락처 |
| 오시는 길 페이지 | /location — 지도, 교통/주차 안내 |
| 사이트 설정 | 관리자 — 회사정보, 지도 URL, 교통안내 입력 |
| VIN 조회 | 차대번호로 차량 정보 조회 → 호환 부품 필터링 |

## 4. 데이터 모델 요약

### 브랜드 (brands/{brandId})
```
name: string          # 브랜드명
category: string      # "import" | "domestic" | "oem"
imageUrl: string?     # 브랜드 카드 이미지 URL (Firebase Storage 또는 외부 URL)
createdAt: timestamp
```

### 상품(products) 필드
```
name: string
description: string
categoryId: string
brand: string?        # 브랜드명
imageUrls: string[]
stock: number
compatibleVehicles: string[]
prices: { individual, business_a, business_b, business_c }
isActive: boolean
isBest: boolean       # 홈 베스트 섹션 노출
isNew: boolean        # 홈 신상품 섹션 노출
createdAt: timestamp
createdBy: string
```

### 사이트 설정 (siteConfig/main)
```
companyName: string
companyAddress: string
companyPhone: string
businessHours: string
businessNumber: string
companyIntro: string
mapEmbedUrl: string       # 지도 iframe URL (네이버 지도 퍼가기)
naverMapUrl: string       # 네이버 지도 공유 링크
kakaoMapUrl: string       # 카카오맵 공유 링크
drivingInfo: string       # 자가용 안내
transitInfo: string       # 대중교통 안내
parkingInfo: string       # 주차 안내
```

## 5. 어드민 메뉴 구성

| 메뉴 | 경로 | 기능 |
|------|------|------|
| 상품 목록 | /admin/products | 전체 상품 조회, 활성화/숨기기, 삭제 |
| 상품 등록 | /admin/products/new | 새 상품 등록 |

| 사이트 설정 | /admin/settings | 회사정보, 오시는 길 정보 입력 |

## 6. 리스크

| 리스크 | 대응 |
|--------|------|
